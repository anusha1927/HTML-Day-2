Explain br tags?

The br tag in HTML document is used to create a line break in a text. It is an empty tag which means it does not need an end tag like < / br >.

What are semantic tags?

			A semantic element clearly describes its meaning to both the browser and the developer.
			Improves readability.

			Examples of non-semantic elements: < div > and < span > - Tells nothing about its content.

			Examples of semantic elements: < form >, < header >, < nav >, < section >, < table >, and < article > - Clearly defines its content.
